# Agricultural APIs for Crop Analysis

## Popular Agro/Plant APIs:

### 1. PlantNet API (Current)
- Plant identification
- Limited to identification only
- CORS issues in browser

### 2. Plant.id API (Recommended)
- **Plant identification**
- **Disease detection**
- **Health assessment**
- **Pest identification**
- Good for agricultural use

### 3. iNaturalist API
- Plant identification
- Community-driven
- Good for educational use

### 4. GBIF API
- Biodiversity data
- Plant occurrence data
- Research-focused

### 5. Agrio API (Agricultural focused)
- **Crop disease detection**
- **Pest identification**
- **Agricultural recommendations**
- **Specialized for farming**

### 6. PestNet API
- Pest identification
- Disease detection
- Agricultural focus

## Best Options for AgriGuru:

### Option 1: Plant.id API
- **URL**: https://api.plant.id/v3/
- **Features**: Disease detection, health assessment
- **Pricing**: Free tier + paid plans
- **Perfect for**: Crop health analysis

### Option 2: Agrio API
- **URL**: https://api.agrio.app/
- **Features**: Agricultural disease detection
- **Specialized**: Farm-specific analysis
- **Perfect for**: Professional farming

### Option 3: Custom Agricultural AI
- **Use**: TensorFlow.js with agricultural models
- **Benefits**: Offline capability, custom training
- **Requires**: Model training and hosting

## Recommendation: Plant.id API
- Best balance of features and accessibility
- Excellent disease detection
- Good documentation
- Reasonable pricing
- Easy integration
